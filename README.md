# Front-end

Aplicativo mobile-first em React + Vite inspirado nas telas de referencia enviadas.

## Rodar localmente

```bash
npm install
npm run dev
```

O Vite abre em `http://127.0.0.1:5173`.

## Telas implementadas

- Inicio com busca, categorias e explicacao de pagamentos.
- Checkout com selecao de Cartao, Pix ou Dinheiro no Local.
- Conclusao de servico por token de 4 digitos ou foto.
- Carteira com saldo, abas e movimentacoes.

## Integracao futura

Configure `VITE_API_URL` apontando para a API Spring Boot quando o back-end estiver publicado ou rodando localmente.
